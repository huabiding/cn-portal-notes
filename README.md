# cn-portal-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容来源于不同时期的信息收集与整理，以静态网页形式存放，便于直接浏览或引用。

## 项目简介

cn-portal-notes 是一个以 HTML 页面为主要载体的归档仓库。每个页面独立存在，不依赖于特定的后端服务或数据库。仓库结构简单，主要面向需要查看或参考这些静态页面的用户。

## 目录说明

仓库根目录下存放了多个 `.html` 文件，文件名通常反映了页面的主题或内容类别。部分页面可能附带相关的资源文件（如图片、样式表），这些资源文件位于对应页面同级的子目录中。

建议通过 GitHub Pages 或其他静态托管服务来浏览这些页面，以获得完整的渲染效果。

## 页面归档说明

- 每个 HTML 页面均为独立归档，内容在发布后通常不再修改，以保留原始信息状态。
- 页面内容可能来源于公开渠道的个人整理，不代表任何官方立场，也不构成任何形式的建议或承诺。
- 部分页面可能包含时间戳或版本标记，用于标识归档时的信息时效。

## 维护说明

- 本仓库不定期更新，新增页面会直接添加到根目录。
- 不保证所有页面的外部链接或嵌入资源长期有效。
- 如发现页面显示异常或内容明显错误，欢迎通过 Issues 反馈，但请注意本仓库不承诺及时修复。
- 不接受涉及具体域名、网站或商业推广的页面提交请求。

## 使用方式

1. 直接点击仓库中的 `.html` 文件，可在浏览器中预览原始源码。
2. 推荐将仓库克隆到本地，或启用 GitHub Pages 功能以获得更好的浏览体验。
3. 如需引用页面内容，请注明来源为本仓库。
